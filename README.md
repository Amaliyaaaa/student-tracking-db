# student-tracking-db
Я писала [статью](https://www.elibrary.ru/item.asp?id=82825883), для которой я реализовывала это приложение.

В ходе проекта я создала систему для отслеживания физической активности студентов. Она объединяет базу данных (на платформе MySQL Workbench) и десктопное python-приложение, разработанное при помощи средств Visual Studio. База данных хранит личные данные студентов, сведения о посещаемости занятий, результаты спортивных тестов и нормативы для разных возрастных групп. Благодаря этому можно сравнивать достижения каждого студента с установленными стандартами. Данные отображаются в виде таблиц на вкладках, можно просматривать, добавлять, редактировать и удалять записи при помощи кнопок. При вводе информации система проверяет её корректность, а все изменения сразу видны на экране. Такое решение значительно облегчило бы работу преподавателей и сотрудников учебного отдела, чтобы они смогли оперативно учитывать посещаемость, следить за выполнением нормативов и анализировать физическую активность студентов без лишней бумажной рутины.

### Как протестировать:
Поскольку база данных создается локально, вам будет недостаточно лишь запустить exe-файл - приложение создано для управления базой данных. Я рекомендую использовать средства, которые я описала, создать базу данных локально на вашем устройстве и при необходимости изменить параметры подключения в py-файле:
```
def __init__(self):
        self.conn = mysql.connector.connect(
            host="localhost",
            user="root",
            password="qwerty1234",
            database="student_activity"
        )
```
