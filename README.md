# student-tracking-db
Я писала [статью](https://www.elibrary.ru/item.asp?id=82825883), для которой я реализовывала это приложение.

В ходе проекта я создала систему для отслеживания физической активности студентов. Она объединяет базу данных (на платформе MySQL Workbench) и десктопное python-приложение. База данных хранит личные данные студентов, сведения о посещаемости занятий, результаты спортивных тестов и нормативы для разных возрастных групп. Благодаря этому можно сравнивать достижения каждого студента с установленными стандартами. Данные отображаются в виде таблиц на вкладках, можно просматривать, добавлять, редактировать и удалять записи при помощи кнопок. При вводе информации система проверяет её корректность, а все изменения сразу видны на экране. Такое решение значительно облегчило бы работу преподавателей и сотрудников учебного отдела, чтобы они смогли оперативно учитывать посещаемость, следить за выполнением нормативов и анализировать физическую активность студентов без лишней бумажной рутины.
